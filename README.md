# Course Notes Compiler

A lightweight Flask web application designed to compile and format video transcripts and reading materials from online courses. This tool formats raw text into a standardized structure, appending it to a single text file to be easily parsed and summarized by AI tools.

## Future Scope
This project is currently a data-entry and formatting tool, with planned future development to evolve into a comprehensive frontend dashboard for reviewing study notes.
